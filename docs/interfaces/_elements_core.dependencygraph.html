Interface DependencyGraph

The type of a plugin dependency graph. This dependency graph represents the reverse dependency relationships of the plugins: a plugin points to the plugins that list it as a dependency. For example, plugin A will point to plugin B if B lists A as a dependency.

The graph is represented as an adjacency list, which itself is represented as an object whose keys are plugin names and whose values are lists of the names of the depending plugins.
